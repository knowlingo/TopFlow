## Advanced Tactics

### Thread Hijacking (Ethical)
When popular accounts tweet about security:
1. Reply with valuable insight first
2. Then mention your tool (if relevant)
3. Don't be spammy

Example:
```

This is why I built my scanner with zero data storage: https://topflow.dev/showcase/security-scanner

No database = no breach surface area.
```

1. Hook (surprising statement)

## Red Flags to Avoid

❌ Don't:
- Buy followers/engagement
- Use automation for replies
- Over-hashtag (max 3 per tweet)
- Tweet the exact same thing daily
- Ignore negative feedback
- Be defensive about criticism

✅ Do:
- Engage authentically
- Share behind-the-scenes
- Celebrate user wins
- Admit mistakes quickly
- Ask for feedback genuinely
- Thank supporters publicly
